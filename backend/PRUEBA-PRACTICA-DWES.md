
# 1.¿Qué endpoint has creado y por qué?

He creado un endpoint que hace que veas todos los pokémons que hay en la base de datos de cada generación.

Lo he hecho por que se requería saber cuantos pokémons se van agregando nuevos por cada generación. Esto ayuda mucho a saber si falta alguno o no, ya qiue te dice el número exacto de pokémons que hay.
